# Support

Thanks for using the Enterprise AutoML Platform! Here's how to get help.

## 📚 Documentation first

- [README](README.md) — overview, installation, quick start
- [docs/](docs/) — architecture, backend, frontend, deployment, API and more
  (run `mkdocs serve` for the rendered site)
- Interactive API reference at `/docs` (Swagger) when the backend is running

## 💬 Questions & discussion

- Open a [GitHub Discussion](https://github.com/MParsa9011/enterprise-automl-platform/discussions)
  for usage questions, ideas and general help.

## 🐛 Bugs & features

- File a [bug report](https://github.com/MParsa9011/enterprise-automl-platform/issues/new?template=bug_report.yml)
  or a [feature request](https://github.com/MParsa9011/enterprise-automl-platform/issues/new?template=feature_request.yml).
- Please search existing issues first to avoid duplicates.

## 🔒 Security

- Do **not** file public issues for vulnerabilities — follow the
  [Security Policy](SECURITY.md).

## 🤝 Contributing

- See [CONTRIBUTING.md](CONTRIBUTING.md) and the
  [development guide](docs/development.md).
